# MLX Hub ⚡️

Mac 유저들을 위한 가장 직관적이고 가벼운 로컬 AI (LLM) 서버 매니저입니다. Apple Silicon (M1/M2/M3/M4) 칩셋의 하드웨어 가속 성능을 100% 활용하여 빠르고 쾌적하게 로컬 AI 모델을 구동할 수 있습니다.

## 📥 다운로드 및 설치

1. 우측의 **[Releases]** 탭에서 최신 버전의 `.dmg` 파일을 다운로드합니다.
2. 다운로드한 `.dmg` 파일을 열고, 안에 있는 `mlx-hub` 아이콘을 **바탕화면이나 `응용 프로그램(Applications)` 폴더로 드래그 앤 드롭**하여 빼냅니다.
   *(주의: dmg 파일 안에서 직접 실행하면 성능 저하가 발생할 수 있습니다.)*
3. 복사한 앱을 실행합니다.

### ⚠️ 설치 시 주의사항
현재 Apple Developer 프로그램에 등록되지 않은 오픈소스 앱이므로, 최초 실행 시 "앱이 손상되었기 때문에 열 수 없습니다"라는 경고가 나타날 수 있습니다. 
이는 macOS의 Gatekeeper 보안 정책 때문이며, 터미널을 열고 아래 명령어를 한 줄 입력하시면 정상적으로 실행됩니다.

```bash
xattr -cr /Applications/mlx-hub.app
```

## ✨ 주요 기능
- **클릭 한 번으로 모델 구동:** 번거로운 터미널 명령어 없이 픽셀 테마의 깔끔한 UI에서 모델을 검색하고 실행하세요.
- **내 Mac 사양에 맞는 스마트 추천:** 내 Mac의 RAM 용량을 실시간으로 분석하여 끊김 없이 돌아가는 최적의 모델 2개를 추천해 드립니다.
- **Apple Silicon 완벽 최적화:** `mxfp8` 등 Mac 하드웨어(NPU) 가속을 100% 활용하는 최신 포맷을 기본 지원합니다.
- **백그라운드 트레이 지원:** 메뉴바 아이콘을 통해 방해받지 않고 언제든 AI 서버 상태를 모니터링하고 제어할 수 있습니다.

## 🤝 외부 앱 연동
MLX Hub는 단순히 모델을 켜는 것에 그치지 않고, 로컬의 다른 앱(예: daddy-keep-working 등)과 연동되어 작동하도록 설계되었습니다. 백그라운드에 켜두기만 하면 연동된 앱들이 알아서 이 서버를 찾아 로컬 AI를 활용합니다.
