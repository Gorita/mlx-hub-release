# MLX Hub ⚡️

Mac 유저들을 위한 가장 직관적이고 강력한 동적 다중 로컬 AI (LLM) 서버 매니저입니다. Apple Silicon (M1/M2/M3/M4) 칩셋의 하드웨어 가속 성능을 100% 활용하여 빠르고 쾌적하게 로컬 AI 모델들을 구동할 수 있습니다.

## 📥 다운로드 및 설치

1. 우측의 **[Releases]** 탭에서 최신 버전의 `.dmg` 파일을 다운로드합니다.
2. 다운로드한 `.dmg` 파일을 열고, 안에 있는 `mlx-hub` 아이콘을 **바탕화면이나 `응용 프로그램(Applications)` 폴더로 드래그 앤 드롭**하여 빼냅니다.
   *(주의: dmg 파일 안에서 직접 실행하면 성능 저하가 발생할 수 있습니다.)*
3. 복사한 앱을 실행합니다.

### ⚠️ 설치 시 주의사항
현재 Apple Developer 프로그램에 등록되지 않은 오픈소스 앱이므로, 최초 실행 시 "앱이 손상되었기 때문에 열 수 없습니다"라는 경고가 나타날 수 있습니다. 
이는 macOS의 Gatekeeper 보안 정책 때문이며, 터미널을 열고 아래 명령어를 한 줄 입력하시면 정상적으로 실행됩니다.

```bash
xattr -cr /Applications/mlx-hub.app
```

## ✨ 주요 기능
- **무제한 모델 동적 구동 (다중 서버 슬롯):** 번거로운 터미널 명령어 없이 여러 개의 서로 다른 모델을 각각 다른 포트(8765, 8766 등)에 병렬로 구동하고 개별적으로 제어할 수 있습니다.
- **카드별 독립형 챗 플레이그라운드:** 실행 중인 각 모델 카드 내부에서 다이렉트로 전용 포트에 라우팅되는 `💬 CHAT TEST`를 실행하여 바로 채팅할 수 있습니다.
- **내 Mac 사양에 맞는 스마트 추천:** 내 Mac의 RAM 용량을 실시간으로 분석하여 끊김 없이 돌아가는 최적의 모델 가이드를 제공합니다.
- **Vision 모델(LMV) 공식 지원:** 백엔드 메타데이터 분석을 통해 Vision 모델 구동 시 대기 지연 안내 및 필요한 패키지(`mlx-vlm`)의 자동 가상환경 관리를 지원합니다.
- **백그라운드 트레이 및 메모리 누수 원천 차단:** 메뉴바 아이콘을 통해 백그라운드 AI 서버 상태를 실시간으로 모니터링할 수 있으며, 좀비 프로세스 방어 시스템이 있어 메인 앱 종료 시 MLX 서버들도 깔끔히 자폭 종료됩니다.
- **초고속 스크롤 성능:** Direct DOM Ref를 사용한 하드웨어 가속 튜닝으로 수많은 모델 목록을 60-120fps 속도로 부드럽게 스크롤할 수 있습니다.

## 🤝 외부 앱 연동
MLX Hub는 단순히 모델을 켜는 것에 그치지 않고, 로컬의 다른 앱(예: daddy-keep-working 등)과 연동되어 작동하도록 설계되었습니다. 다중 포트로 구동되는 여러 서버 상태를 Unix Domain Socket(`/tmp/mlx-hub.sock`)을 통해 실시간으로 발행하므로, 연동된 외부 앱들이 필요한 모델 포트를 찾아 자유롭게 로컬 AI를 활용할 수 있습니다.
